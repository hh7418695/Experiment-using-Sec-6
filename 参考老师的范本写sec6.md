## 6. Psychophysical Experiment（心理物理实验）
本章节详细描述用于评估受试者对虚拟可变形物体力学属性感知能力的心理物理实验设计与实施方案。
### 6.1 Participants（受试者）

将从我身边的朋友中招募 10 名健康受试志愿者（总人数10人，其中男性7名，女性3名；年龄范围 18-25岁之间.所有受试者均为右撇子，并且自述无神经系统疾病、运动功能障碍或严重视力问题。实验方案将遵守学院关于人体研究的伦理规范。

（计划是10名受试者，对于受试者的个人信息比如专业年龄男女，有没有参与过类似触觉感知实验的经验，后面再讨论，这里暂时无法确认）

---

### 6.2 实验装置

受试者通过桌面型力反馈装置（Geomagic Touch / TouchX，3 自由度位置输入，3 自由度力输出）与虚拟可变形物体交互。装置放置在受试者面前的桌面上，受试者使用优势手（右手）握持触控笔进行操作。

底层触觉渲染由 C++ 程序实现，伺服频率约为 1 kHz，通过官方 SDK 与力反馈设备通信。  
在每个伺服周期中，当前笔尖位置发送至 Python 仿真模块；仿真模块基于质量–弹簧–弯曲模型更新虚拟条带状态，并返回需要呈现的反作用力，由 C++ 端输出到设备末端执行器。

在正式实验阶段，受试者佩戴眼罩以及Apple Airpods Pro3(支持降噪)耳机。耳机中播放白噪音，用以遮挡视觉信息及设备、键盘产生的环境噪声。  
实验员坐在一侧，负责口头提示、控制程序运行，并在终端界面输入受试者的选择。受试者看不到任何屏幕提示，只能通过力反馈设备感知物体的动态响应，并回答实验者二选一的问题。

---

### 6.3 刺激

#### 6.3.1 虚拟可变形物体模型

所有刺激均基于同一类虚拟可变形物体：一条一维虚拟“香肠”（以下称呼都不加双引号）。 
该条带由 5 个点质量串联而成，相邻质点之间通过拉伸元件连接，并在相邻线段之间加入弯曲元件以表示抗弯刚度。各质点质量、间距和阻尼参数在所有条件中保持不变。



香肠的一端在虚拟空间中固定，另一端通过虚拟耦合与触觉笔尖相连。当受试者移动笔尖时，虚拟第一个node点被带动，香肠发生拉伸和弯曲，仿真根据当前形变计算反作用力并反馈到手柄。

模型中使用两个关键参数描述物体的力学属性：一个是拉伸刚度 \(k_s\)**，决定香肠在长度方向被拉长时的阻力大小；另一个是**弯曲刚度 \(k_b\)**，决定香肠在发生弯曲时的抗弯硬度。除这两者外，其他物理参数在所有刺激条件下保持固定。

#### 6.3.2 刚度参数设置（10 个虚拟物体）

实验中共定义 10 个虚拟物体，通过在给定范围内系统调节拉伸刚度和弯曲刚度得到。未缩放的刚度取值如下：(目前我已经在在代码中写死了刚度数值范围，如有必要，为了方便研究的数据，可重新布局数值范围，目前仅用数值如下)

- 拉伸刚度列表：  
  `stretching_val_list = [50, 287.5, 525, 762.5, 1000]`
  
  ps：每间隔237.5，共5个值
- 弯曲刚度列表：  
  `bending_val_list = [0, 0.025, 0.05, 0.075, 0.1]`

出于数值稳定性考虑，上述数值在仿真内部可能会乘以一个常数因子（例如1000），但本文中仅报告其概念上的原始范围。

基于上述列表构造两个 5 水平的刚度连续体：
- **object 1–5：拉伸刚度连续体（Stretching set）**  
k_s范围={50, 287.5, 525, 762.5, 1000}，弯曲刚度 \(k_b = 0.05\) 固定不变  
- **object 6–10：弯曲刚度连续体（Bending set）**  
k_b范围={0, 0.025, 0.05, 0.075, 0.1}，拉伸刚度 \(k_s = 525\) 固定不变  

因此，object 1–5 在弯曲刚度恒定条件下构成一个拉伸刚度连续体；object 6–10 在拉伸刚度恒定条件下构成一个弯曲刚度连续体。

### 6.4 实验设计

本研究采用两备择强迫选择（two-alternative forced choice, 2AFC）的心理物理实验范式，包括两个 block：

1. **拉伸 block（Stretching block）**：仅改变拉伸刚度 \(k_s\)，弯曲刚度 \(k_b\) 固定；
2. **弯曲 block（Bending block）**：仅改变弯曲刚度 \(k_b\)，拉伸刚度 \(k_s\) 固定。

在两个 block 中，受试者每个试次都会依次接触两个虚拟object，并根据当前 block 的指令和提问内容，在二者之间做出“第一个”或“第二个”的二选一判断。

#### 6.4.1 拉伸 block

在拉伸 block 中，刺激取自object 1–5。  
此时弯曲刚度 \(k_b = 0.05\) 保持不变，自变量为拉伸刚度 \(k_s\)。

对这 5 个object的所有**无序配对**进行测试，共得到10种不同的物体组合。每个物体配对重复 4 次，因此拉伸 block 共包含40个试次。
在对同一配对的每次重复中，两个object的呈现顺序（先后）以程序已经在代码库中设置的random函数随机出现，约等于50%概率决定，以避免人类触觉感知的顺序偏差。

#### 6.4.2 弯曲 block

在弯曲 block 中，刺激取自object 6–10。  
此时拉伸刚度 \(k_s = 525\) 固定不变，自变量为弯曲刚度 \(k_b\)。

同样对 5 个object的所有无序配对进行测试，共 10 个不同配对，每个配对重复 4 次，因此弯曲 block 也包含40个试次。

呈现顺序的随机化方式与拉伸 block 相同。两个 block 的整体顺序（先做拉伸还是先做弯曲）可在受试者间进行平衡。

### 6.5 实验流程

#### 6.5.1 训练阶段

在正式实验开始前，受试者先在不戴眼罩和耳机的情况下完成若干练习试次，以熟悉任务和操作方式。

实验员向受试者说明：每个试次中将依次接触两个虚拟物体——“第一个”和“第二个”，并根据提示判断哪一个在某一维度上更“硬”：

- 拉伸 block 中的提问问题为：  
  “哪一个更不容易被拉长？也就是更紧、更难拉伸。”
- 弯曲 block 中的提问问题为：  
  “哪一个在弯的时候感觉更硬？”

在练习期间，实验员可根据受试者回答情况提供适当反馈，确保其理解“第一个 / 第二个”的含义以及回答方式。当受试者能够稳定地根据指令回答“第一个”或“第二个”后，进入正式实验阶段。

#### 6.5.2 正式实验阶段

正式实验中，受试者佩戴眼罩和Apple Airpods pro3耳机，耳机中播放宽带噪声以屏蔽视觉信息和设备噪音。  
此时受试者无法看到屏幕上的信息，也听不到键盘声音；实验流程由实验员通过终端控制并进行口头提示。

每个试次的流程如下（适用于两个 block）：

1. **试次初始化**  
   控制程序从预生成的 trial 列表中选取当前试次，包含：  
   - 所属 block 类型（stretch / bend）  
   - 物体配对（参考物体 ref 与比较物体 comp）  
   - 随机化后的先后顺序（first_object / second_object）

2. **第一刺激时段**  
   实验员提示：  
   > “现在请感受第一个物体。”  

   程序将当前仿真对象切换为 `first_object`。受试者通过移动触觉笔尖对虚拟条带进行探索，时间约为 5–8 秒：  
   - 在拉伸 block 中，引导其沿拉伸方向拉动，感受被拉长时的阻力；  
   - 在弯曲 block 中，引导其通过侧向移动使条带弯曲，感受弯曲时的硬度。

3. **间隔期**  
   仿真逐渐将输出力减小至接近 0，受试者短暂放松。  
   实验员提示：  
   > “第一个结束，请准备第二个。”

4. **第二刺激时段**  
   程序将当前仿真对象切换为 `second_object`，受试者再次探索约 5–8 秒。

5. **作答**  
   实验员提醒受试者做出判断：  
   
   - 拉伸 block：  
     > “哪一个更不容易被拉长？”  
   - 弯曲 block：  
     > “哪一个在弯的时候感觉更硬？”  
   
   受试者口头回答“第一个”或“第二个”，实验员在终端中按 `1`（表示选择 first_object）或 `2`（表示选择 second_object）进行记录。

6. **休息与继续**  
   每个 block 内试次按上述流程依次进行。  在 拉伸block点40次trial结束后，会安排短暂休息，以缓解疲劳，待休息完成进入弯曲block点实验。

### 6.6 数据记录与分析

#### 6.6.1 行为数据记录

每个trial结束后，控制程序会自动向对应的 CSV 文件写入一行行为数据。 
在拉伸 block 中，拉伸刚度 \(k_s\) 较大的物体被视为在“更不容易被拉长”这一维度上的正确答案； 
在弯曲 block 中，弯曲刚度 \(k_b\) 较大的物体被视为“在弯曲时更硬”的正确答案。程序据此计算 `is_correct`字段。

#### 6.6.2 计划分析

对于每位受试者和每个 block，将对各物体配对计算正确率。  
可根据刚计算的 `is_correct` 字段，将配对分为“较小差异”（如相邻刚度水平）与“较大差异”（非相邻刚度水平）两组，分别统计这两类条件下的平均正确率。

随后在受试者间计算每种条件的均值和标准误，用以描述群体水平的辨别表现。  
在条件合适时，可以将正确率视为刚度差的函数，拟合心理物理曲线，估计辨别阈值（如 JND）。

最后，将比较拉伸 block 与弯曲 block 的整体表现以及小差异条件下的敏感度，以判断在本实验给定的刚度区间内，受试者对拉伸刚度变化敏感还是对弯曲刚度变化更加敏感。
